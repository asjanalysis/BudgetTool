# BudgetTool

Single-page web tool for turning a budget spreadsheet into a consolidated PDF report. Upload your Excel budget, attach invoices and proofs for each expense, then download a combined PDF that includes the expense details and any supporting documents. All processing happens in your browserâ€”no data is stored or uploaded to a server.

## Usage
1. Choose the budget template version (Expenses tab for v1, Personnel/Non-Personnel tabs for v2).
2. Upload the Excel budget file and click **Load expenses** to parse it.
3. For each expense, optionally attach an invoice and a proof of payment (PDF, PNG, or JPG).
4. Click **Generate report PDF** to download the consolidated report.
